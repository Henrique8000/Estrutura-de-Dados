# 📚 Data Structures in Java

This repository contains my studies and practical implementations of **Data Structures** using **Java**.

## 📖 Overview

The goal of this project is to strengthen my understanding of core data structures and algorithms through hands-on practice.

Most exercises were developed throughout my **Information Systems degree**, along with additional self-directed practice to strengthen my problem-solving and algorithmic skills.

## 🧩 Topics Covered

- Linked Lists (Singly & Doubly)
- Stacks (LIFO)
- Queues (FIFO)
- Trees
- Searching Algorithms
- Sorting Algorithms
- Additional exercises and implementations

## 🛠️ Tech Stack

- **Language:** Java

## 🎯 Objectives

- Build a solid foundation in data structures  
- Improve algorithmic thinking and problem-solving  
- Write efficient and organized code  
- Prepare for technical interviews
